About the project

A modern take on the aging online poker platform: new game variants, ad-free online games for desktop & mobile using React animations and django-channels websockets.

Implemented bleeding-edge technology, including an event stream system, MOVE-like architecture, pure-functional reactive frontend animations (to do this we had to write our own frontend animations library), and an in-house system for time-travel debugging and introspection tools.
